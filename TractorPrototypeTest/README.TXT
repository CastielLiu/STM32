BKP_DR2 最低字节为A -> 转角传感器偏移值已校准
BKP_DR2 次低字节为A -> 转角传感器参数已设定 最大AD值，最大转角
BKP_DR3 存放转角传感器AD偏移值 -> 方向盘中位时的AD值
BKP_DR4 存放转角传感器电压值经AD转换所得的最大AD值; 硬件最大值，并非被限制后的值
BKP_DR5 存放转角传感器最大角度值(一般为整数); 硬件最大值，并非被限制后的值 《最高位为方向》
